# 9. Дебагинг

## Инструменты отладки
| Инструмент | Назначение |
|------------|------------|
| Chrome DevTools | Отладка фронтенда |
| VS Code Debugger | Пошаговая отладка |
| React DevTools | Инспекция компонентов |
| Console.log | Быстрая отладка |

## Основные подходы

### Фронтенд
```javascript
// Отладка через console
console.log('debug:', variable);
console.table(array);
console.trace('trace');

// Точка останова в коде
debugger;
```

### Бэкенд
```javascript
// Логирование
logger.debug('Событие:', data);
logger.error('Ошибка:', error);
```
