# 4. Требования к дизайну — «АлИИна» чат-виджет

## Стиль и концепция

**Визуальная концепция:** Современный, минималистичный и премиальный чат-виджет, ассоциирующийся с качественной мебелью и профессиональным сервисом. Дизайн должен вызывать доверие, быть ненавязчивым и удобным при встраивании на сайт vstroy-mebel.ru.

**Ключевые принципы:**
- **Доверие и профессионализм** — чистые линии, аккуратная типографика, сдержанные цвета
- **Ненавязчивость** — виджет не перекрывает контент сайта, легко сворачивается
- **Живое общение** — плавные микро-анимации, индикаторы активности, аватар АлИИны
- **Премиальность** — мягкие тени, стеклянные эффекты (glassmorphism), плавные градиенты
- **Простота** — минимум визуального шума, фокус на содержании диалога

**Образ АлИИны:**
- Образ: дружелюбная, профессиональная, женственная
- Аватар: стилизованная иконка-аватар с натуральными тонами (не фото, не мультяшный)
- Тон: тёплый, но деловой

## Компоненты UI

### 1. Кнопка-иконка виджета (WidgetToggle)

- **Закрытое состояние:** Круглая кнопка 60px в правом нижнем углу с аватаром АлИИны или иконкой чата
- **Тень:** `0 4px 16px rgba(0, 0, 0, 0.15)`
- **Hover:** Масштабирование `scale(1.08)` + усиление тени
- **Анимация появления:** `fadeInUp` при загрузке страницы (задержка 2 сек)
- **Бейдж:** Зелёная точка «онлайн» (8px) в правом верхнем углу

### 2. Заголовок виджета (Header)

```
┌─────────────────────────────────────┐
│ 🟢 АлИИна                    ─  ✕  │
│    онлайн                           │
└─────────────────────────────────────┘
```

- Фон: Primary (`#8B6F47`) или плавный градиент Primary → Primary Dark
- Аватар АлИИны: 36px, скруглённый
- Статус: «онлайн» — зелёная точка + текст
- Кнопки: свернуть (─) и закрыть (✕), белые, 32×32px
- Высота: 60px
- Радиус скругления: 16px 16px 0 0

### 3. Область сообщений (ChatWindow)

- Фон: Background (`#FAFAF7`)
- Автопрокрутка вниз при новом сообщении
- Плавный scroll (`scroll-behavior: smooth`)
- Скроллбар: тонкий, стилизованный (4px, цвет `#D0D0D0`)

## Анимации и переходы

| Элемент | Анимация | Длительность | Easing |
|---------|----------|-------------|--------|

| Индикатор «печатает» | 3 точки с пульсацией (scale 0.6↔1) | 1.2s | `ease-in-out`, infinite |

| Кнопка отправки (появление) | `scale(0→1)` + `rotate(-90°→0)` | 200ms | `ease-out` |
| Аудио-волна (TTS) | Анимация equalizer-баров | 800ms | `linear`, infinite |
| Прогресс загрузки файла | `width(0%→100%)` | Зависит от размера | `linear` |

## Адаптивность

### Mobile (320–767px)
- Виджет открывается на **весь экран** (fullscreen overlay)
- Скругления: 0
- Заголовок: фиксированный сверху
- Поле ввода: фиксированное снизу
- Область сообщений: занимает всё пространство между
- Кнопка-иконка: 56px, отступ 16px от края
- Клавиатура: поле ввода поднимается над виртуальной клавиатурой

### Tablet (768–1023px)
- Плавающее окно: 380×560px, правый нижний угол
- Отступ от края: 20px
- Все элементы как на Desktop

### Desktop (1024px+)
- Плавающее окно: 380×560px, правый нижний угол
- Отступ от края: 24px
- При resize окна — виджет сохраняет свою позицию

### Ландшафтная ориентация (mobile)
- Виджет: уменьшенная высота, прокрутка сообщений
- Поле ввода: остаётся фиксированным снизу

## Тематизация (кастомизация)

Виджет поддерживает кастомизацию через CSS-переменные при встраивании на сайт:

```css
:root {
  --aliina-primary: #8B6F47;
  --aliina-primary-dark: #6B5335;
  --aliina-secondary: #F5EDE3;
  --aliina-accent: #C9A96E;
  --aliina-bg: #FAFAF7;
  --aliina-surface: #FFFFFF;
  --aliina-text: #2D2D2D;
  --aliina-radius: 16px;
  --aliina-font: 'Inter', system-ui, sans-serif;
  --aliina-width: 380px;
  --aliina-height: 560px;
  --aliina-position-right: 24px;
  --aliina-position-bottom: 24px;
}
```

Это позволяет заказчику адаптировать цвета виджета под дизайн сайта без изменения исходного кода.

## Звуковое оформление

| Событие | Звук | Описание |
|---------|------|----------|
| Новое сообщение от АлИИны | Мягкий «дзинь» | Короткий, ненавязчивый (200ms) |
| Начало записи голоса | Тихий клик | Подтверждение начала записи |
| Окончание записи | Тихий клик (высокий) | Подтверждение отправки |
| Ошибка | Мягкий двойной тон | Не агрессивный |

- Все звуки опциональны и отключаются в настройках
- Громкость: 30–50% от системной
- Формат: WAV или OGG Vorbis (< 10 КБ каждый)
