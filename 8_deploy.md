## Процесс деплоя

### Автоматический деплой
```bash
# При пуше в main ветку деплой запускается автоматически
git push origin main
```

### Ручной деплой
```bash
# 1. Сборка проекта
npm run build

# 2. Деплой
npm run deploy
```
